Technologies

zEdit uses the following technologies:

  • ES6 JavaScript - Programming Language
  • NodeJS - JavaScript Runtime
  • Electron - Platform
  • AngularJS 1.5 - JavaScript Framework
  • SASS - CSS Extension Language
  • CSS3 - Stylesheet Language
  • HTML5 - Markup Language

APIs

zEdit offers a number of APIs which can be developers can tap into to extend the application's functionality.

Guides

This documentation includes several development guides which cover several ways in which developers can contribute to the zEdit project.
