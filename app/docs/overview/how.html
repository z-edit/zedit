1. Download

Download the latest release of zEdit from GitHub.

2. Installation (Setup)

If you downloaded the setup executable, run it. A shortcut to start zEdit will be placed on your desktop once it completes.

2. Installation (Portable)

If you downloaded the portable archive, extract it to the folder where you would like to install zEdit.

Note: multiple installations of zEdit is unnecessary and strongly discouraged. zEdit allows you to choose the application and game mode from the Start View.

3. Running zEdit

Run the executable or double click a shortcut. If you're using Mod Organizer you will need to run zEdit through MO the same as any other application.

4. Profile selection

You will now be presented with the Start View. Profiles should be automatically created for your installed games. If there isn't a profile for the game you want to use zEdit with you can create it from the Profiles Modal. Select the profile you want to use and click the Start Session button.

5. Specifying your load order

You will now be presented with the Load Order Modal. Specify the load order you want to use and click the OK button.

6. Editing

The loader will display as your plugins are loaded. Once the loader is done you will be pesented with the Edit View, from which you can view and edit your plugin files.

7. Saving your changes

You can save your changes at any time by pressing Ctrl + S to open the Save Modal.

8. Closing the program

You can close the program by clicking the X in the upper right hand corner of the window. If there are any changes to your plugin files you will be prompted to save them before the program is closed.
